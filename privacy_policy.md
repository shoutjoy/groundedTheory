# [근거 이론 코딩 어시스턴트] 개인정보처리방침 (Privacy Policy)

본 개인정보처리방침은 **[근거 이론 코딩 어시스턴트]**(이하 '본 프로그램')가 사용자의 개인정보를 처리하는 방식과 보안에 대해 설명합니다. 본 프로그램은 사용자의 연구 데이터를 최우선으로 보호하며, 관련 법령 및 Chrome 웹 스토어 개발자 정책을 준수합니다.

---

### 1. 개인정보 수집 및 처리 항목
본 프로그램은 서비스 제공을 위해 필요한 최소한의 데이터에만 접근하며, 별도의 서버로 수집하거나 전송하지 않습니다.
* **클립보드 데이터**: 사용자가 NotebookLM의 답변을 '저장' 또는 '스크랩'할 때 해당 텍스트를 읽기 위해 일시적으로 접근합니다.
* **텍스트 콘텐츠**: 사용자가 직접 입력하거나 수정한 단계별 프롬프트 및 분석 결과물.
* **브라우저 스토리지 정보**: 앱의 설정 상태 및 로컬에 저장된 분석 데이터.

### 2. 데이터 처리 방식 (로컬 저장 원칙)
본 프로그램은 사용자의 데이터를 외부로 유출하지 않는 것을 원칙으로 합니다.
* **외부 전송 금지**: 처리된 모든 데이터는 개발자나 제휴사를 포함한 제3자에게 전송되지 않으며, 외부 서버와의 통신을 수행하지 않습니다.
* **로컬 저장소 활용**: 모든 데이터는 사용자의 브라우저 로컬 저장소(`chrome.storage.local`) 내에만 저장됩니다.

### 3. 데이터 사용 목적
본 프로그램은 오직 다음과 같은 목적으로만 데이터를 사용합니다.
* Google NotebookLM 서비스 내에서의 근거 이론(Grounded Theory) 분석 단계(개방/축/선택 코딩) 자동화 지원.
* 분석 결과의 단계별 관리, 통합 보기 및 마크다운(Markdown) 파일 내보내기 기능 제공.

### 4. 데이터 보유 및 파기
* **보유 기간**: 데이터는 사용자의 브라우저 내에만 존재하며, 별도의 보관 기간을 두지 않습니다.
* **파기 방법**: 사용자가 크롬 확장 프로그램 관리 페이지에서 본 프로그램을 **삭제(제거)**하면, 로컬 저장소에 기록된 모든 데이터는 즉시 영구 파기됩니다.

### 5. 사용자의 권한
* 사용자는 언제든지 프로그램 내의 기능을 통해 저장된 데이터를 수정하거나 삭제할 수 있습니다.
* 사용자는 본 프로그램이 요구하는 권한(클립보드, 저장소 등)을 거부할 권리가 있으나, 이 경우 핵심 기능 이용이 제한될 수 있습니다.

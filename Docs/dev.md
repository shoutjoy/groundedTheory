이 뷰어팝업을  1단계부터 새창보기 옆에 동그라미버튼을 만들어서 누르면 이것으로보게 해줘 

# 뷰어 설계 / 개발 
1. 뷰어 팝업 (답변 스크랩 모아보기 / 합쳐서 보기)
이동: 제목 바를 드래그하면 팝업 위치를 옮길 수 있습니다.
리사이즈: 오른쪽 아래 모서리 리사이즈 핸들(↘ 아이콘)을 드래그해 크기 조절 가능 (최소 너비 280px, 높이 200px).
보기 버튼: 클릭 시 원문 ↔ 마크다운 렌더링 전환. 다시 누르면 "원문"으로 되돌아갑니다.
md 저장 버튼: 현재 내용을 .md 파일로 다운로드합니다. (파일명: 제목-타임스탬프.md)
닫기: 팝업을 닫습니다.
"합쳐서 보기", "답변 스크랩 모아보기"를 누르면 위 뷰어 팝업이 뜨며, 모두 같은 방식으로 이동·리사이즈·보기·md 저장이 가능합니다.

2. 새창보기 옆 ○ 버튼 (이것으로 보기)
dev.md 레이아웃에 맞춰 각 행에 ○ 버튼을 두었습니다.

행	새창보기	○ (이것으로 보기)
1단계	mdproviewer 새 탭	1단계 저장 내용을 뷰어 팝업으로 표시
2단계	mdproviewer 새 탭	2단계 저장 내용을 뷰어 팝업으로 표시
3단계	mdproviewer 새 탭	3단계 저장 내용을 뷰어 팝업으로 표시
합쳐서 보기	합친 내용을 mdproviewer로	합친 내용을 뷰어 팝업으로 표시
답변 스크랩 모아보기	스크랩 전체를 mdproviewer로	스크랩 전체를 뷰어 팝업으로 표시
새창보기: 기존처럼 mdproviewer.vercel.app 새 탭으로 열고, 내용 전달.
○: 같은 내용을 우리 뷰어 팝업으로만 엽니다. (드래그·리사이즈·보기·md 저장 가능)
4번째 행에는 "합쳐서 보기" 옆에 새창보기와 **○**를 모두 넣었습니다.
○ 버튼에 마우스를 올리면 툴팁 "이것으로 보기"가 표시됩니다.
